# Guacamole for Moodle #

## Current version ##

1.1

## Features ##
- Add virtual classroom, provided by Apache Guacamole in Moodle

## Supported languages ##
- english, french

## Installation ##

- Copy files into the following directory **/mod/guacamole/** and visit **/admin/index.php** in your browser

## Requirement ##

Moodle 3.9 or greater.
